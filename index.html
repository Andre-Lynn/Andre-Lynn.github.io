CSS: SELECTORS


CSS Selector Code Selector Name Explanation
* Universal Selector Selects all HTML elements for CSS styling
. Class Selector Selects HTML class elements for styling. (Note: it is possible to add more than one class name to HTML element).
# ID Selector Selects a single HTML element with a unique ID.
[attribute] Attribute Selector Selects all HTML elements with the said attribute.
HTML element:hover, :focus, :visited, :disabled, :active Pseudo-Class Selector Selects pseudo-classes, i.e. HTML elements with a certain user action, for styling. E.g.: whenever mouse hovers over a paragraph, the paragraphâ€™s background color turns to lime.
, Comma Allows to select multiple elements via multiple selectors, e.g. h1, special
Specificity: ID overrides class overrides type
Chaining: combining selectors is possible. E.g. h1.special {...} will select only h1 elements which are classed as ""special".
